# T-008: Remove search hint and make title dynamic

## Goal

- Remove the static search hint text from the search controls.
- Make the browser page title follow the current dataset reference date instead of a hardcoded date.

## Scope

- Update the static HTML markup so the search hint is no longer shown by default.
- Update the client script so the title is derived from `payload.reference_date`.
- Keep the visible reference date in the hero synced with the same data source.

## Verification

- Run `python scripts/agentctl.py task lint`.
- Load the page and confirm the search hint is gone.
- Confirm the browser title uses the current `reference_date` from `assets/active_lotteries.json`.
