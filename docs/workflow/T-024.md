# T-024: Refresh active lotteries for 2026-04-10

## Goal

- Refresh the published lottery dataset so the site reflects the active permits as of `2026-04-10`.
- Remove permits that are no longer active by the reference date.
- Add any newly active permits from the latest official VID registry workbook.
- Recheck incomplete or weakly sourced enrichment entries and fill missing public details from verified public sources.

## Scope

- Obtain the latest official VID workbook exposed on `https://www.vid.gov.lv/lv/loterijas`.
- Recompute the active permit set for `2026-04-10`.
- Update `assets/active_lotteries.json` to match the current active permit set.
- Update `assets/enriched_info.json` for newly active permits and for active permits whose public details are missing, incomplete, or stale.
- Record the added, removed, and rechecked permits plus the public sources used.

## Verification

- `python -m json.tool assets/active_lotteries.json`
- `python -m json.tool assets/enriched_info.json`
- `.venv/bin/python scripts/extract_lotteries.py --date 2026-04-10 --input "<latest official VID workbook>" --check`
