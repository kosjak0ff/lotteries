# T-003: Normalize line endings in site files

## Goal

- Convert accidental CRLF endings back to LF in the published site files without changing code or markup.

## Scope

- Update `index.html`.
- Update `assets/app.js`.
- Update `assets/style.css`.
- Preserve file contents exactly.

## Verification

- `git diff --exit-code -- index.html assets/app.js assets/style.css`
