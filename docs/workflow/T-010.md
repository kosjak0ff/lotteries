# T-010: Refresh active lotteries for 2026-03-28

## Goal

- Refresh the published lottery dataset to match the latest IAUI registry for the reference date `2026-03-28` while preserving existing on-site details for permits that remain active.

## Scope

- Download the latest IAUI lottery registry XLS published on the official IAUI page.
- Recompute the active permit set for `2026-03-28`.
- Update `assets/active_lotteries.json` by removing inactive permits and adding newly active permits without rewriting unchanged active entries unnecessarily.
- Update `assets/enriched_info.json` only for newly added permits, filling missing public details from verified public sources and checking registration dates.

## Implemented

- Queried the official IAUI registry page and confirmed its metadata is updated as of `26.03.2026`.
- Recomputed the active permit set for `2026-03-28` from the local workbook `Loteriju reģistrs_16.03.2026..xls`, which already contains entries starting on `2026-03-24` and `2026-03-27`.
- Updated `assets/active_lotteries.json` from reference date `2026-03-23` to `2026-03-28`.
- Removed inactive permits `8795` and `8830`, and added newly active permits `8931` and `8953`.
- Preserved the total published count at `43`, while updating changed workbook values for permits `8755` (`end` moved to `2026-04-09`) and `8840` (`end` moved to `2026-04-07`).
- Kept `assets/enriched_info.json` unchanged because no reliably verifiable public source could be confirmed for the newly active permits from this environment without inventing details.

## Notes

- The official IAUI download endpoint exposed on the registry page was reachable enough to confirm page metadata, but repeated attempts to download the binary attachment from this environment stalled before a file could be saved. Because of that, this refresh uses the local workbook already present in the repository root.

- `python -m json.tool assets/enriched_info.json`
