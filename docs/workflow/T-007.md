# T-007: Refresh active lotteries from latest IAUI registry

## Goal

- Refresh the published lottery dataset to match the latest IAUI registry for the reference date `2026-03-23` while preserving existing on-site details for permits that remain active.

## Scope

- Download the latest IAUI lottery registry XLS published on the official IAUI page.
- Recompute the active permit set for `2026-03-23`.
- Update `assets/active_lotteries.json` by removing inactive permits and adding newly active permits without rewriting unchanged active entries unnecessarily.
- Update `assets/enriched_info.json` only for newly added permits, filling missing public details from verified online sources and checking registration dates.

## Implemented

- Fixed `scripts/extract_lotteries.py` so it can parse the current IAUI workbook date columns stored as day-first strings (for example `23.03.2026`) and so blank text cells no longer serialize as the literal string `nan`.
- Refreshed `assets/active_lotteries.json` for reference date `2026-03-23` while preserving all 39 previously active permit records byte-for-byte and only appending the four newly active permits: `8795`, `8830`, `8943`, and `8955`.
- Reintroduced permit `8830` because it is active again for `2026-03-23` in the IAUI registry, despite having been removed earlier for the prior site snapshot.
- Added verified enrichment for the newly active permits based on current public sources:
  `8795` (official Trentini lottery landing page),
  `8830` (official campaign page + rules PDF),
  `8943` (official campaign page + rules/permit PDF).
- Added a partial enrichment record for `8955` with the official BAVA site URL and an explicit note that no separate public lottery rules page could be found as of `2026-03-23`, so unverified fields were not invented.

## Verification

- `.venv/bin/python scripts/extract_lotteries.py --date 2026-03-23 --input "Loteriju reģistrs_16.03.2026..xls" --check`
- `python -m json.tool assets/active_lotteries.json`
- `python -m json.tool assets/enriched_info.json`
- Compared `HEAD:assets/active_lotteries.json` with the refreshed file to confirm that the 39 previously active permits stayed unchanged, with only `8795`, `8830`, `8943`, and `8955` added and no permits removed.
