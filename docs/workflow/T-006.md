# T-006: Automate IAUI source sync

## Goal

- Automatically discover the latest IAUI lottery registry XLS, download it into the repository, regenerate the static site dataset, and keep the published site metadata in sync.

## Scope

- Add a sync script for the IAUI source page and XLS download.
- Update dataset generation so source metadata can be published with the JSON payload.
- Update the frontend to render source metadata dynamically.
- Add a scheduled GitHub Actions workflow for automatic sync and commit.

## Implemented

- Added `scripts/sync_iaui_source.py` to discover the latest IAUI download link, fetch the XLS into a stable repository path, and regenerate the JSON dataset when the source changes.
- Added `.github/workflows/sync-iaui.yml` with both `workflow_dispatch` and a daily scheduled run.
- Updated `scripts/extract_lotteries.py` so generated JSON can include source metadata (`source_label`, `source_updated_at`, `source_url`).
- Updated `index.html` and `assets/app.js` so the site title, reference date, and source label are rendered dynamically from the JSON payload.
- Pushed the implementation commit to GitHub after upgrading the repository credential to a PAT with `workflow` scope.

## Verification

- `python -m py_compile scripts/extract_lotteries.py scripts/sync_iaui_source.py`
- `test -f .github/workflows/sync-iaui.yml`
