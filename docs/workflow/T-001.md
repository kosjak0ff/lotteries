# T-001: Active Latvian lotteries 2026-03-20 (RU site)

## Goal

- Extract lotteries that are active on 20.03.2026 from `Loteriju reģistrs_16.03.2026..xls`, excluding magazine/newspaper related ones.
- Produce machine-readable JSON with normalized fields and a Russian-language static site to display them.

## Scope

- Data extraction script and resulting JSON under `assets/`.
- Static site (HTML/CSS/JS) with Russian UI, showing table, counts, and search.
- No backend; client-side only.

## Verification

- `python scripts/extract_lotteries.py --check`
- `python -m json.tool assets/active_lotteries.json`
